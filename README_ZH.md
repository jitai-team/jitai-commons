## ✨ 核心特性

### 后端能力

- **计算组件**: 数学计算、文本处理、日期时间、逻辑运算、统计分析等函数
- **常量与枚举**: 业务字典、错误码、数据库字段类型等标准定义
- **全局变量**: 时间相关、周期时间、用户相关变量
- **日志记录**: 统一的日志记录接口

### 前端能力

- **计算组件**: 与后端一致的计算函数（数学、文本、日期、逻辑）
- **变量管理**: 应用变量、全局变量管理
- **数据处理**: 数据转换、格式化等工具
- **UI 反馈**: 消息提示、确认对话框、加载状态
- **消息通信**: 组件间消息传递
- **公共工具**: 文件处理、通用工具函数
- **异常处理**: 统一的异常捕获与处理
- **多线程支持**: Web Worker 封装
